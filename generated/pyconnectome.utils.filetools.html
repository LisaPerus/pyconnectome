pyconnectome.utils.filetools.fslreorient2std(input_image, output_image, fslconfig='/etc/fsl/5.0/fsl.sh')[source]¶

Reorient an image to match the approximate orientation of the standard template image (MNI152).

It only applies 0, 90, 180 or 270 degree rotations. It is not a registration tool. It requires NIfTI images with valid orientation information in them (seen by valid labels in FSLView). This tool assumes the labels are correct - if not, fix that before using this. If the output name is not specified the equivalent transformation matrix is written to the standard output.

The basic usage is:
fslreorient2std <input_image> [output_image]
Parameters:

input_image: str (mandatory)

The image to reorient.

output_image: str (mandatory)

The reoriented image.

fslconfig: str (optional, default DEFAULT_FSL_PATH)

The FSL configuration batch.
